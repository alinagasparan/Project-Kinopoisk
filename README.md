Пользователь имеет возможность:
-войти/зарегистрироваться в систему
-отфильтровать самостоятельно фильмы
-написать помощнику и получить от него ответ
-написать комментарий-отзыв о фильме
-поставить отметку на фильм "запланирован/просмотрен" и увидеть списки по этим отметкам
-посмотреть новости кино и примьеры

Администратор имеет право:
-добавить новости о кино
-добавить фильм в базу данных
-модерировать комментарии

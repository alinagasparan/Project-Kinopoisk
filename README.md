# Project-Kino
Проект включает в себя создание сайта с кино со встроенным ии помощником который выдает фильмы на основе ввода текста с пожеланиями пользователя.

Подразделяем сайт на страницы

Главная страница:

Название сайта, плашка поиска по названию, кнопка перехода в разделы

-Премьеры фильмов, сериалов

-Кнопка помощника

-Новости в сфере кино

-Кнопка перехода в личный кабинет


Нажимая на кнопку перехода в разделы:

Каталог(База данных):

-Фильмы

-Сериалы

Бот-ассистент

Профиль


Личный кабинет:

Аватар, Никнейм, Список просмотренных/запланированных кино


Каталог:

Фильмы как пример:
Сразу выдает список всех фильмов с изображением, справа фильтры по всему чему только можно, жанры, год, кол-во серий и т. д

В зависимости от переключения фильтров меняется список фильмов. 


Помощник:

Чат с ботом:

Задаем ему задачу найти фильм по таким то интересам
Он выдает нужные фильмы


Окно фильма:

-Афиша фильма 

-Все данные о нем

-Пункт просмотрен/запланирован 

-Комментарии пользователей

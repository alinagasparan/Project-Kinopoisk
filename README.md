# Project-Kino
Пользователь должен иметь возможность:

•регистрации/входа в систему

•посмотреть премьеры/новости кино

•посмотреть всю информацию про кино

•отфильтровывать кино в каталоге

•ввести запрос помощнику и получить ответ

•возможность оценивать фильм

•возможность написать комментарий-отзыв

•возможность отметить фильм как запланированный/просмотренный и посмотреть эти списки в ЛК

Администрация имеет возможность:

•добавлять новости кино

•модерировать комментарии на сайте

•добавлять новые фильмы/сериалы в БД

# Project-Kino
Мы должны иметь возможность:

•регистрации/входа в систему

•посмотреть премьеры фильмов

•посмотреть фильм и всю информацию про него

•отфильтровать фильмы

•запрос помощнику

•возможность оценивать фильм

•возможность написать комментарий-отзыв

•возможность отметить фильм как запланированный/просмотренный
